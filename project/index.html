❤️ Life Expectancy Project ❤️

An interactive website that displays a few data visualizations on a life expectancy dataset. In particular, its main goal is to show, country by country, how this has varied over the years, including looking at what variables have influenced it. ⚕️ Life Expectancy Dataset ⚕️

General problems on the dataset:

As data pre-processing I uploaded the dataset from the provided website.

Since some column names were bad formatted, I had to correct them. Next I analyzed the entrues of 'Country' column in order to check if there were any invalid namens that needed to be removed but this was not the case. However, I had to change some entries in this column in favor of using more common names to identify some states. This was very helpful to me in the implementation of the choropleth map.

At this point I was ready to select data for individual graphs. The process was the same for all: selecting the 'Country' and 'Year' columns, selecting the column of interest to show on the current graph, sorting the resulting rows, and removing null values.

Also, for each graph, I had to reformat the data using 'Country' as a pivot, thus going to have, instead of the other two columns, a new column for each different value of 'Year' with, as value, the value that the third column had for that 'Country'-'Year' pair.

Slightly different discussion regarding the ridgechart as the columns, excluding 'Country' and 'Year', were two and no longer one. To keep the data formatting the same, I simply added a new 'Type' column indicating which of the two columns the mapped value referred to. For the heatmap instead, I calculated the correlation index between variables with a script.

Finally, for the implementation of the map, using a script, I linked my dataset with a JSON file that would draw the map in order to add properties containing the dataset data to it.

Line Charts

Visualizing, for each country, the life expectancy and adult mortality along the years (2000-2015), respectively.

Line Charts

The first graph is the line chart which, when a country is selected, shows how life expectancy for that country (y-axis) has varied over the years (x-axis). Such a chart is useful when one wants to visualize effectively how, especially with regard to poorer countries, the trend appears to be upward.

The second graph, on the other hand, is the line chart that, when a country is selected, shows how adult mortality for that country (y-axis) has varied over the years (x-axis). In this case, thanks to such a graph, we can see how the trend tends instead toward a slight decrease along the years.

Bar Chart

This chart, when a year is selected, allows the display of a comparison of infant mortality values in that year for up to 10 countries. Through it, it is possible to highlight differences in the quality of life in different states.

Ridge Chart

Next graph is the ridgechart that aims to show BMI and HIV/AIDS distributions, computed using Kernel Density Estimation (KDE), of the selected countries (max 10) considering all the years stored in the dataset.
For each country, the lighter area represents the BMI distribution while the darker area represents the HIV/AIDS distribution. This allows for a more generic comparison of BMI and HIV/AIDS values than that obtained by looking at individual years.

Pass the mouse hover chips to isolate the country from the others

Heatmap

For each state, it is shown an heatmap describing the correlation between each selected variable. This chart is useful to understand how, for each state, certain variables such as, in our case, adult mortality, infant mortality, BMI, HIV/AIDS and GDP are affected by each other.

Choropleth Map

Last graph is the choropleth map that, when a year is selected, aims to visualize the life expectancy value for each country. You can mouse over a single state to obtain information about its exact value for that year. As you can read from the legend beside, states for which we don't have any information about are mapped with the grey color.
